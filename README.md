# AI Chatbot using Python (NLTK)

This project is a rule-based chatbot built using Python and Natural Language Toolkit (NLTK).

## Features:
- Responds to greetings (hi, hello, hey)
- Handles basic user queries
- Uses pattern matching techniques
- Simple NLP-based interaction

## Technologies Used:
- Python
- NLTK

## How to Run:
1. Install nltk:
   pip install nltk

2. Run the chatbot:
   python chatbot.py
